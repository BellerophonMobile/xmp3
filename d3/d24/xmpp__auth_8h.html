Handler for the inital stream start to begin authentication.

This should be the first handler defined on a new XML stream.

Precondition:
Assumes that this is the first handler called. The client should send a starting <stream> tag.
Postcondition:
On success, if not authenticated, proceeds to SASL authentication, else begins resource binding. On failure (parse error, etc.), closes the connection and stops parsing.
Parameters:
stanzaThe parsed stanza
parserThe XML parser instance.
dataAn xmpp_client instance.
